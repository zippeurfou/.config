# GitHub Push Checklist

Before pushing your dotfiles to GitHub, verify the following:

## ‚úÖ Privacy & Security

- [ ] Check `.zprivate` is in `.gitignore` (it should be)
- [ ] No API keys, tokens, or passwords in any tracked files
- [ ] No company-specific secrets in tracked configs
- [ ] Check `opencode/command/` for any sensitive documents
- [ ] Review `nvim/` config for any personal info

## ‚úÖ Files to Review

These files might contain personal information:

- `opencode/command/*.md` - May contain work-specific notes
- `nvim/LSP_ANALYSIS.md` - May contain project-specific details
- `zsh/.zalias` - Contains AWS/GitHub configurations (review lines 56-124)

## ‚úÖ Files That Won't Be Pushed (Already in .gitignore)

- `.zprivate` - Your secrets file
- `.zcompdump` - Generated file
- `gcloud/`, `nvm/`, `op/` - Local tool directories
- `atuin/key` - Atuin encryption key
- `zsh/.zsh_sessions/` - Session data

## üîß Optional: Clean Up Personal References

You might want to replace or remove:

1. **In `zsh/.zalias`:**
   - `create_project_function()` - Uses your GitHub account
   - `opencode_setup()` - AWS SSO with specific profile
   - Consider making these configurable via `.zprivate`

2. **In `zsh/.zshrc`:**
   - Hardcoded path: `/Users/mferradou/Developer/lua` (line 134)
   - Google Cloud SDK path in Downloads folder (line 149-152)
   - Consider using `$HOME` instead of hardcoded username

3. **In `ghostty/config`:**
   - Path to tokyonight theme (line 1) - uses your username

## üìù Recommended: Add to .zprivate Template

Create a `.zprivate.example` file with:

```bash
# Example .zprivate file
# Copy to .zprivate and fill in your values

# GitHub
export GITHUB_TOKEN="ghp_your_token_here"

# OpenAI
export OPENAI_API_KEY="sk-your_key_here"

# AWS Profile (if different from 'data')
export AWS_PROFILE="your_profile"

# Any other personal secrets...
```

## üöÄ Ready to Push

Once you've reviewed everything:

```bash
cd ~/.config
git add .
git commit -m "Add dotfiles configuration"
git remote add origin https://github.com/yourusername/dotfiles.git
git push -u origin main
```

## üí° Tips

- Consider making sensitive paths configurable
- Use `${HOME}` instead of `/Users/mferradou` where possible
- Keep work-specific configs in `.zprivate`
